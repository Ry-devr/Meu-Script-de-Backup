# Meu scipt de backup hyprland
Meu objetivo com esse codigo é criar um script de backup em python, sei q um script feito em sh pode ser melhor para meu objetivo mais python por enquanto é a linguagem que tenho umpouco mais de experiencia. 

## Requisitos:
- Python3.14 ou superior
- Funciona apenas em linux, Principalmente no arch ou seu derivados que utiliza o gerenciado "pacman"
- Nao utilizar dois pendrives na hora do backup o comando ainda nao tem tratamento para essa variaçao, por enquanto

## Funçoe:
- faz backup das configs. Se quiser pode mudar os diretorios que seram comprimidos na parte do codigo onde esta comentado "COMPRIMIR"
- faz backup dos pacotes instalados

## Comando
#### fazer backup

```python3.14 backup.py```

#### instalar o backup
```python3.14 install_backup.py```
